# 🌟 Astrology CRM System

A full-stack CRM system for managing clients, appointments, and consultations for astrology services.

---

## 🚀 Features

- User Authentication (JWT)
- Client Management (CRUD)
- Appointment Scheduling
- Consultation Tracking
- Multi-user data isolation
- Secure backend APIs

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- React (if frontend)
- JWT Authentication

---

## 🔐 Security

- Each user can only access their own data
- Protected routes using JWT middleware
- Data isolation using userId

---
